::: CIRCULATION SETTING :::

Maximum Borrowing Parameter
This indicates maximum number of books a Patron type can borrow. After a patron reaches max number of borrow, system will generate a warning while issuing.

Allowable Overdue
Maximum number of books a Patron type can overdue. System generates warning while issuing if a parton exceeds this limit.

Fixed Borrowing Due
This field indicates a fixed due date for a patron. For example, to allow faculty patron type to borrow a book for whole semester enter the last date of semester for faculty patron type.

Ceiling Due Date
This date is the closing date of library. A patron must return all library material before this date. Circulation can't issue a book exceeding this date. For example, closing of a semester is Jan' 31, enter a date earlier to Jan'31 for a parton to return all library material on of before this date.

Maximum Fine
This field indicates the maximum amount a patron can accumulate in fines. If parton exceeds this limit, System gives a warning to circulation. If amount is set to zero, System generates warning for any fine amount.

Maximum Reservation
Maximum reservation a patron can hold. System generates warning if a parton exceeds this limit. Circulation can override this limit while reserving a book for a patron.

Reservation Priority
Default priority set by system while reserving a book. Circulation can change this priority while reserving for a patron.

Loan Period
The number of days or hours that a member of the patron type can borrow a copy belonging to the specific circulation type. If the Loan Period is "hours," the settings in Grace should be "hours". If the Loan Period is "fixed," Grade should be in days and the due date is determined by the Due date set in Fixed Borrowing Due.

Grade Period
Number of days/ hours a patron is not charged fine after overdue for a particular circulation.

Max Renew
Max. Number of renews a patron is allowed for a particular circulation.

Fine Increment
Amount indicated is the amount charged per day or per hour. Per day or per hour is specified in Loan period. If loan period is hours, fine is per hour, if loan period is per day, fine is per day and if loan period is fixed, fine is per day.

Fine Max. Implemented
Maximum amount a patron can be charged for a damaged, lost, or overdue copy for a circulation type.
