CHANGE STUDENT'S CRITICAL INFORMATION - HELP

Change Student ID:
Click to change student ID to a new ID. Student ID can't be changed to a new ID if new ID already exists.

Change Student Entry Status Information
Click to change student's entry status incase it is not correct. Student's entry status may be <tranferee>, <New>, <Second Course>. If the student is a second courser, please create the student with previous course and proper entry status(new/transferee/second course), and the follow the 4th link in this page to create and entry for second course.

Change Student Entry Information (Other)
Click this to change the student's entry information other than Entry status. The following informations can be changed. Click corresponding EDIT ICON to change the information. Any of the student's information can't be changed if student is having grade information for entry for School year,term.

1. Entry Status: It is explained on top
2. Course/Major: Changes the entry course/major/curriculum year information.
3. School year/term: Click to change school year information
4.Year Level : Click to change student's year level entry.

Create Student's Change Course Information:
To encode the grade for change course student's, first create the student with information prior to change course information. After encoding the grade for previous course, Click this link to change the course information. Enter new course, major, student's curriculum year, School year and Term. After creating change course information, Student's grade can be encoded for the new course. In grade sheet encoding, System should be able to display student's course for a spcified school year/ term. For example 2003-2004/1st sem course of student is BSBA and 2003-2004/2nd sem course is BSCS.

Change Student's Year level Information
Click to change the student's year level information. It lists all the year level of the student after entering student's ID. System does not allow to delete any year level entry if student is having grade information for that school year-term/year level

Manage Student's Course,SY/SEM Information
 Displays and allows to delete or add new entry for student's curriculum ,SY/SEM information. If student is having grade information, delte is not allowed. Usually, Click this link only if it is needed to view the student's curriculum history, or if it is need to create an entry for grade sheet information encoding. For example, Student is a change courser and Student is having grade information for the changed course and does not have information for previous school year, Create new entry information for student for the specified school year.
